# Car Store

## Api/Crud for a MongoDB database
* Built with Node.js and Express
* Api documented with Swagger Ui
* Virtualized with Docker

### Development
* Development environment
    * `docker-compose up`
* Linting and formatting
    * `npm run lint`
    * `npm run format`
* Testing
    * `npm run test`

### End points
* `/cars`
* `/api-docs`
